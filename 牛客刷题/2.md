# 计算字符个数(C++)
**题目**
>写出一个程序，接受一个由字母和数字组成的字符串，和一个字符，然后输出输入字符串中含有该字符的个数。不区分大小写。
```c
#include<iostream>
#include<string>
using namespace std;
int main()
{
    string str;
    char one;
    int count=0;
    cin>>str;
    cin>>one;
    int lens=str.size();
    for(int i=0;i<lens;i++)
    {
        if(str[i]==one)
        {
            count++;
        }
        if(str[i]==one+32)
        {
            count++;
        }
        if(str[i]==one-32)
        {
            count++;
        }
    }
    cout<<count<<endl;
}
```
